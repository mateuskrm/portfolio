Serviços/Habilidades

Flutter

Flutter é um framework de desenvolvimento de aplicativos móveis nativos multiplataforma que usa uma única base de código e oferece recursos avançados como widgets personalizáveis e compilação AOT, usando a linguagem de programação Dart.

Dart

Dart é uma linguagem de programação moderna, orientada a objetos e de tipagem forte, projetada para desenvolvimento de aplicativos móveis, web e desktop. Ela oferece recursos como compilador JIT (Just-In-Time), suporte a programação assíncrona e uma ampla biblioteca padrão.

Oracle Database

Oracle Database é um SGBD relacional da Oracle, muito utilizado em empresas devido à sua confiabilidade, escalabilidade e recursos avançados como segurança, replicação de dados e suporte a diversas linguagens de programação.
